About SmartBooks

An intelligent online library recommendation system powered by Machine Learning.

What is SmartBooks?

SmartBooks is an online book recommendation platform designed to help users discover books based on their interests, favorite books and personal ratings.

The system combines a modern web interface with Machine Learning techniques to generate personalized recommendations and improve the reading experience.

Project Purpose

The main goal of this project is to design and implement a functional recommendation system for an online library using Python, Flask, SQLAlchemy, Bootstrap and Machine Learning algorithms.

📚

Users can browse books, search by title or author and filter books by category.

Recommendation Method

SmartBooks uses a Content-Based Filtering approach. Each book is represented using textual information such as title, author, category and description.

TF-IDF Vectorization converts book text into numerical vectors, while Cosine Similarity measures how similar two books are. Books with higher similarity scores are recommended to the user.

Why This System?

Online libraries often contain many books, making it difficult for users to find the most relevant titles. A recommendation system improves navigation and helps users discover books that match their interests.

This project demonstrates how Machine Learning can be integrated into a real web application to provide personalized and practical results.

Master Thesis Project

SmartBooks was developed as part of a Master of Science thesis in Applied Computer Science, focusing on the design and implementation of a Machine Learning based recommendation system for an online bookstore.
