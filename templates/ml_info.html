Machine Learning Information

Explanation of the recommendation algorithm used in SmartBooks.

Content-Based Filtering

SmartBooks recommends books by analyzing the content of books that a user has added to favorites. The system compares book title, author, description and category to find similar books.

TF-IDF Vectorization

TF-IDF transforms textual book information into numerical vectors. This helps the system understand which words are important in each book description.

Cosine Similarity

Cosine Similarity measures how close two book vectors are. A higher similarity score means that two books are more related and one may be recommended to the user.

Recommendation Process

  1. The user adds books to favorites.
  2. The system extracts book content features.
  3. TF-IDF converts text into vectors.
  4. Cosine Similarity compares books.
  5. The most similar books are recommended.

Algorithm Summary

Recommendation Type Content-Based Filtering
Vectorization Method TF-IDF
Similarity Metric Cosine Similarity
Dataset 100 books from multiple categories
Used Features Title, Author, Description, Category
